# Product Overview

KAZE KEZA Portfolio is a modern, minimalist portfolio website showcasing data storytelling and creative technology work. The site emphasizes performance, accessibility, and sustainable web development practices.

## Core Features

- Custom p5.js animations (loading, spiral backgrounds, hero animations)
- Glass morphism UI with subtle interactions
- Dark/light theme with system preference detection
- Blog and project showcase with Sanity CMS integration
- Data visualization components using Recharts
- Contact form with validation
- Full accessibility (WCAG 2.1 AA compliant)

## Design Philosophy

Follows Dieter Rams' principle "Weniger, aber besser" (Less, but better):
- Clean, minimal, purposeful design
- Honey accent color (#EBA937) as primary brand color
- Organic animations and micro-interactions
- Mobile-first responsive approach

## Performance Goals

- 30fps cap for animations
- Lazy loading and dynamic imports
- Reduced motion support
- Minimal bundle size through tree-shaking
- Optimized for Vercel deployment
